# **App Name**: FundChain

## Core Features:

- Wallet Connection: Allow users to connect their wallets using RainbowKit or Web3Modal.
- Campaign Creation: Enable users to create crowdfunding campaigns with a title, description, funding goal, and deadline; all stored on-chain.
- Campaign Listing: Display all campaigns in a clean, modern UI.
- Donation to Campaign: Allow users to donate ETH or tokens to any campaign.
- Withdraw Funds: Enable the campaign creator to withdraw funds if the funding goal is reached and the deadline has passed.
- Refund Donations: Allow donors to refund their donations if the funding goal is not reached by the deadline.
- AI-Powered Campaign Assistant: Utilize an AI tool to provide campaign creators with smart recommendations and estimated conversion rates to help ensure their campaigns succeed. 

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to evoke trust and stability in the platform.
- Background color: Light grey (#F5F7FA), a soft and neutral background that will emphasize content and the bright primary.
- Accent color: Vibrant cyan (#00FFFF) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a sans-serif for clean readability.
- Use modern, minimalist icons to represent different campaign categories.
- Clean and futuristic design with a focus on ease of use and visual appeal.
- Subtle animations for transactions and UI interactions.
